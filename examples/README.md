# Examples
This directory contains example scripts that demonstrate how to use the **PdfPageManipulator** API to manipulate PDF files. The examples are self-contained and require no additional explanation.

## Getting started

Follow these steps to run the examples:

### 1. Install dependencies
Run the installation script:
```bash
./install.sh
```
This script installs the tools required to run the examples in this directory.

### 2. Generate examples
Run the example generation script:
```bash
./run.sh
```
This script executes the Python example scripts and generates PDF files that demonstrate the results.

### 3. Clean up
Remove all generated PDF files:
```bash
./cleanup.sh
```
This script removes all PDF files generated by the examples that end with the suffix `_PPMTest.pdf`.

## Notes

- Run `install.sh` once during the initial setup.
- Run `run.sh` whenever you want to regenerate the example PDFs.
- Use `cleanup.sh` to remove all PDFs created by `run.sh`.
